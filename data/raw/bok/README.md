## 한국은행 데이터 다운로드 방법

사이트: https://ecos.bok.or.kr → 통계검색

| 파일명 | 검색어 | 주기 | 기간 |
|---|---|---|---|
| base_rate.xlsx | 기준금리 | 월 | 2013.01~2025.12 |
| mortgage_rate.xlsx | 주택담보대출금리 | 월 | 2013.01~2025.12 |
| household_debt.xlsx | 가계대출 | 월 | 2013.01~2025.12 |
| m2.xlsx | M2 광의통화 | 월 | 2013.01~2025.12 |
| bsi_realestate.xlsx | 기업경기실사지수 부동산업 | 월 | 2013.01~2025.12 |

### 다운로드 방법
1. 검색어 입력 후 통계 선택
2. 조회기간 설정 (2013.01 ~ 2025.12)
3. 엑셀 다운로드 버튼 클릭
