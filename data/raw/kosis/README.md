## 통계청 데이터 다운로드 방법

사이트: https://kosis.kr → 국내통계 → 주제별 통계

| 파일명 | 경로 | 기간 |
|---|---|---|
| cpi.xlsx | 물가 → 소비자물가조사 | 2013~2025 월별 |
| income_median.xlsx | 소득소비자산 → 가계동향 | 2013~2025 연별 |
| population.xlsx | 인구 → 주민등록인구통계 | 2013~2025 월별 |

### 다운로드 방법
1. 해당 경로 탐색 후 통계표 클릭
2. 기간 및 지역(서울) 필터 설정
3. 다운로드 → 엑셀 선택
